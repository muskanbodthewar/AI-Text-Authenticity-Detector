# 🤖 AI Text Authenticity Detector

A Machine Learning web application that classifies text as **Human Written** or **AI Generated** using Natural Language Processing (NLP) and a Logistic Regression model.

---

## 🚀 Features

- Detect AI-generated text
- Human vs AI prediction
- Confidence score
- Prediction probabilities
- Word and character count
- Interactive Streamlit web interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- Joblib
- Pandas
- NumPy
